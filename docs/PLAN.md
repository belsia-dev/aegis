# PLAN

## 2026-04-07

### 요청
- 프로젝트를 `.deb` 패키지로 만들기
- APT 저장소에 올릴 수 있는 형태로 배포 준비하기

### 확인한 사항
- 현재 저장소에는 `debian/` 디렉터리나 Debian 패키징 스크립트가 없음
- 현재 저장소에는 `docs/` 디렉터리도 없음
- systemd 서비스 파일 `aegis.service`는 존재함

### 작업 계획
1. Debian 패키지 구조와 설치 대상 경로를 확정한다.
2. `debian/` 패키징 파일 또는 빌드 스크립트를 추가해 `.deb` 생성이 가능하게 만든다.
3. 서비스 파일, 설정 파일, 설치 후 동작을 패키지 설치 흐름에 맞춘다.
4. 로컬에서 패키지 빌드 또는 빌드 절차 검증을 수행한다.
5. APT 저장소 업로드 절차를 문서로 정리한다.
6. 작업 종료 시 `docs/CONTEXT.md`와 `docs/TODO.md`를 현재 상태에 맞게 갱신한다.

### 현재 상태
- `debian/` 패키징 파일 추가 완료
- `.deb` 빌드용 스크립트 추가 완료
- APT 저장소 게시 가이드 문서 추가 완료
- 사용자 빌드 로그 기준 `.deb` 생성 확인
- 패키징 경고 제거 후 재빌드 필요
